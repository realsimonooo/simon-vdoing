## 概念

/ `Regular Expression`对象 用于匹配字符串中字符组合的模式。

​	用来检索、替换符合某个模式的文本； 表单、过滤词、联想词…

​	

## 在JS的应用

### 创建正则表达式对象

``` javascript
// 利用 RegExp new对象
var regexp = new RegExp( /表达式/ );

// 通过 字面量创建
var rg = /表达式/;
```

### 检测表达式的正确性

利用`test()`正则表达式的对象方法即可检测字符串是否符合规则：

```javascript
var rgtest = new RegExp(/表达式/);
rgtest.test(str);
// 检测str文本是否符合rgtest的规则
```



## 正则表达式特殊字符

| 规定字符             |                                          |
| -------------------- | ---------------------------------------- |
| ^                    | 匹配文本的行首                           |
| $                    | 匹配文本的行尾                           |
| /^ $/                | 精准匹配某个文本                         |
| []                   | 只要包含一系列的字符中其中的某个字符就行 |
| ^[]$                 | 只能有其中的**一个**字符                 |
| -                    | 范围符                                   |
| /^[a-z]$/            | 只能有26英文字母任何一个字母才返回true   |
| /^[a-zA-z0-9_-]$/    | **字符组合**,限定其中的字符任一；        |
| `/^[^a-zA-z0-9_-]$/` | `^`表示取反，在方括号内部                |



| 量词符 | 用于设定某个模式==出现的次数==                           |
| ------ | -------------------------------------------------------- |
| /^a*$/ | >=0 出现0次或者很多次                                    |
| /^a+$/ | >=1 出现1次或者很多次                                    |
| /^a?$/ | 0次或1次                                                 |
| {n}    | 重复n次                                                  |
| {n,}   | 重复n次或更多次                                          |
| {n,m}  | 重复n次到m次 （规定文本字数）// 逗号后不要有空格否则无效 |

```javascript
var reg = /^[a-zA-Z0-9]{6,16}$/
// 规定其中检测的次数6-16次
```



+   括号总结
    +   小括号() 表示优先级；
    +   方括号 [] 表示字符集合，匹配括号中的任意字符；
    +   大括号 {} 表示量词符，重复的次数；

## 参数

```javascript
/ 表达式 /[switch]
```

`switch` 有三种值：

+   `g` : 全局匹配；
+   `i`：忽略大小写；
+   `gi`：全局匹配+忽略大小写；
