# 对象

+   对象是一组无序的相关属性和方法的集合；如`String，Number, Arrays, Function`；
+   对象由属性和方法组成。



+   使用对象可以是表达结构跟清晰：应用于储存信息时，属性与数值的对应；

## 创建对象

三种方式：

+   利用**字面量**创建对象

    +   `{}` 内包含表达这个具体事物的属性和方法 （值键对的形式，值：可以是**任意属性**）；

        ```javascript
        var person = {
            name : 'Simon',
            age : 19
           	selfintro : function(){
                alert('Hello~');
            } // 这是一个方法（位于对象以内的函数），不需要声明；
        }
        
        ```

        

+   利用**`new Object()`** 创建对象

    ```javascript
    var person = new Object();
    // 通过 .来设定属性;
    person.name = 'Simon';
    person.age = 19;
    person.selfintro = function(){
        alert("Hello~");
    }
    ```

+   利用**构造函数**创建对象

    ```javascript
    function Person(name, age) {
    	this.name = name;
    	this.age = age;
    	this.selfintro = function() {
    	alert('Hello, my name is ' + this.name);
    }
        
    var Simon = new Person('Simon',19);
        Simon.selfintro();
    ```

    +   约定该构造函数名**==首字母大写==**；

对象的调用： 

+   普通属性可以使用 `.`或者 `[]`

    ```javascript
    person.name;
    person['name']
    ```

+   调用方法属性时：

    ```javascript
    person.selfintro();
    // 注意方法后面必须要加括=号
    ```



## 遍历对象属性

使用`for...in语句`对对象/数组进行循环遍历操作：

```javascript
for(变量 in 对象名称){
    // 执行...
}
```
